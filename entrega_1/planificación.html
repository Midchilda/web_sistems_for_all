Planificación

    La planificación del proyecto se llevó a cabo de la siguiente forma:

  • Análisis

    El primer paso que hemos dado ha sido analizar las distintas maneras de crear el proyecto, basándonos en bocetos a lápiz que hemos ido realizando hasta dar con una interfaz aceptable a nuestra manera de ver la aplicacion.

  • Referencias

    Hemos trasladado los botetos a código html y css para poder conseguir una interfaz gráfica a nuestro gusto. Para ello, también hemos consultado diferentes páginas de hospitales y centros médicos para tener una guía y para poder comprender el uso de estas páginas

  • Detalles

    Se detalla con minuciosidad las necesidades tanto básicas como, más adelante, unas más amplias que deberá cubrir nuestro proyecto, así como su funcionalidad. Así mismo, se detallará en que tipos de dispositivos podrá funcionar y el contenido en general que queremos transmitir con la página.

  • Diseño

    Creacion de bocetos para obtener una idea básica del proyecto final que queremos conseguir.

  • Distribución

    Se distribuye de forma equitativa el trabajo. Se proponen diferentes reuniones para llevar un control y ver como avanza cada miembro del equipo, además de para resolver dudas en conjunto.

  • Implementacion del proyecto

    Cada miembro del equipo realiza la parte que le haya sido asignada logrando la apariencia adecuada a la imagen que queremos reflejar. En las reuniones se hará una puesta en común de como va tomando forma la aplicación.
